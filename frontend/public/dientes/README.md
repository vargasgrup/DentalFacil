# Imágenes dentales realistas (PNG)

Coloca aquí los PNG con fondo transparente. El odontograma los carga automáticamente;
si falta un archivo, usa un dibujo anatómico procedural (SVG) como fallback.

## Nomenclatura

```
{FDI}_{vista}.png
```

Ejemplos:
- `11_vestibular.png`
- `36_vestibular.png`
- `36_oclusal.png`
- `26_lingual.png`
- `default.png` (opcional, fallback genérico)

Vistas: `vestibular` | `lingual` | `oclusal`

## Dimensiones

512×512 o 1024×1024 px, RGBA, fondo transparente.
